"Space War" adalah game sederhana bertema pertempuran luar angkasa yang dikembangkan menggunakan bahasa pemrograman Python dalam mata kuliah Pemrograman Berorientasi Objek. Pemain mengendalikan sebuah pesawat luar angkasa untuk menembak dan menghindari serangan dari musuh yang datang secara bertahap. Game ini memanfaatkan prinsip OOP (Object-Oriented Programming) seperti penggunaan class, inheritance, dan encapsulation dalam pengelolaan elemen game seperti pemain, peluru, dan musuh. Proyek ini bertujuan untuk melatih logika pemrograman serta penerapan konsep OOP dalam pembuatan aplikasi interaktif.
